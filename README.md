# webblog
自己的博客

前端使用React技术栈，使用create-myapp脚手架搭建

服务端使用Java SpringBoot

功能包括：
* 文章上传
* 文章列表展示
* 文章内容展示（支持markdown）
* 个人信息展示及友链
